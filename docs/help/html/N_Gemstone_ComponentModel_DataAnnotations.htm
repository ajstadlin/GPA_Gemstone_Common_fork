Gemstone.ComponentModel.DataAnnotations Namespace

Contains classes and attributes used for component modeling used with data annotations.
Classes
  ClassDescription
AcronymValidationAttribute
Represents a RegularExpressionAttribute for validating Acronyms, i.e., expressions that consist only of upper case letters, numbers, '!', '-', '@', '#', '_' , '.' or '$'.
EmailValidationAttribute
Represents a RegularExpressionAttribute for validating e-mail addresses.
InitialValueScriptAttribute
Defines an attribute that will define an initial value script for a modeled table field that will get evaluated and assigned in the target use environment, e.g., Javascript.
IsDeletedFlagAttribute
Defines an attribute that will define the field name that represents a record marked for deletion.
LabelAttribute
Defines an attribute that will define a UI label to used for a modeled table field.
PrimaryLabelAttribute
Defines an attribute that will define the field name that best represents the row identification to a user.
UrlValidationAttribute
Represents a RegularExpressionAttribute for validating URL's.
