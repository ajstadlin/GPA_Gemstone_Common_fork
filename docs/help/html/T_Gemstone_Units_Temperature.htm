Remarks
This class behaves just like a Double representing a temperature in Kelvin; it is implicitly castable to and from a Double and therefore can be generally used "as" a double, but it has the advantage of handling conversions to and from other temperature representations, specifically Celsius, Fahrenheit, Newton, Rankine, Delisle, Réaumur and Rømer.
Example
This example converts Celsius to Fahrenheit:
C#
Copy
public double GetFahrenheit(double celsius)
{
    return Temperature.FromCelsius(celsius).ToFahrenheit();
}
