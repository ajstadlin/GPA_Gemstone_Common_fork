Remarks

Example to set initial value to 30 days from now using a Javascript expression: [InitialValueScript("(new Date()).addDays(30)")]

Note that the DefaultValueAttribute should be used to set any constant values for new modeled record instances and the gemstone.expressions.model.DefaultValueExpressionAttribute should be used for applying any needed server-side run-time expression defaults, either of these will get assigned when using the gemstone.data.model.TableOperations<T>.NewRecord function. The InitialValueScriptAttribute is used to initialize the property value using an expression that gets evaluated in the target environment.
